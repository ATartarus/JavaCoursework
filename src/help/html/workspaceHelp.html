Компоненты рабочей области

1. Блок метаинформации

Содержит поля для ввода основной информации о зачёте. Для добавления элементов в выпадающее меню необходимо нажать кнопку "+". При вводе в текстовые поля информации, не соответствующей шаблону, поле обводится красным цветом, а при нажатии клавиши Enter выводится сообщение об ошибке.

2. Блок информации о группе

Содержит таблицу с информацией о студентах и их результатах. Для добавления/удаления записи о студенте используются соответственно кнопки "+" и "-".

3. Блок дополнительной информации

Содержит дополнительную информацию, вычисляемую на основе предыдущих блоков.
